# Doc Quality Gate Pack

This pack defines the machine-readable contracts, manifests, policies, suites, and waiver model behind `npm run docs:check`.

Use this pack when the repository needs to answer:

- which docs are governed
- which contract applies to each doc class
- which issues are blocking versus warning-level
- which waivers exist and when they expire
- where the deterministic governance report should be written

The durable rationale for these rules lives under `docs/technical/doc-governance/`.
