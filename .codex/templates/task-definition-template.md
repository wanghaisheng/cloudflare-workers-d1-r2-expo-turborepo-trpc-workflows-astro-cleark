## 🎯 使用指南

### 1. 创建任务定义
```bash

cp .codex/templates/task-definition-template.md tasks/新任务.md

# 填写任务定义
# 按照模板填写所有必需字段
```

### 2. 验证任务定义
```bash
# 运行验证脚本
python .codex/scripts/task_definition_detector.py tasks/新任务.md

# 检查验证结果

```
